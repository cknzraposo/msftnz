# 01 – Prerequisites

Before you dive in, make sure you have everything on this list ready. Each item links to the guide that walks you through it.

---

## ✅ Checklist

Work through this list from top to bottom. Each item is covered in a dedicated guide.

| # | Item | Why you need it | Guide |
|---|---|---|---|
| 1 | **GitHub account** | Store and collaborate on your code | [02 – GitHub Setup](02-github-setup.md) |

## 💻 System Requirements

| Requirement | Minimum |
|---|---|
| Operating System | Windows 10/11, macOS 12+, or Ubuntu 20.04+ |
| RAM | 8 GB (16 GB recommended) |
| Disk space | 5 GB free |
| Internet connection | Required throughout |

---

## 🔑 Accounts You'll Need to Create

You will need to sign up for the following (all free for students):

1. **GitHub** — [github.com/signup](https://github.com/signup)
2. **Microsoft Azure** — Free account with $200 credit: [azure.microsoft.com/free](https://azure.microsoft.com/en-us/free/)
   - If your university provides Azure for Students access, use that link from your institution instead.

> **💡 Tip:** Use your **university email address** when signing up for both GitHub and Azure — this may unlock additional free benefits (GitHub Student Developer Pack, Azure for Students credits).
